# Phase 2: Zentropy (Quantum-Guard)

Welcome to **Phase 2: Zentropy**. Building upon the quantum-safe foundations laid in Phase 1, Zentropy expands into a more holistic suite of next-generation security tools, emphasizing the convergence of Post-Quantum Cryptography (PQC) and Zero-Knowledge (ZK) proofs.

## Documentation

For a comprehensive guide on getting started, architectural details, and using the complete Zentropy suite, please visit our official documentation site:
- 📖 [**Zentropy Documentation**](https://zentropy-docs.vercel.app)

## Project Structure

This directory serves as the core codebase for Phase 2. Below is an overview of the primary components:

- **`pqc_backend/`**: The core backend infrastructure enabling post-quantum cryptography operations.
- **`quantum_wallet_ui/`**: The frontend user interface for the Zentropy quantum-safe wallet.
- **`zk_prover/`**: Components responsible for generating and handling Zero-Knowledge proofs.
- **`starknet_contracts/`**: Smart contracts tailored for integration with the Starknet ecosystem.
- **`scripts/`**: Development, deployment, and operational utility scripts.
- **`docker/` & `Dockerfile`**: Containerization settings for seamless service deployment.
