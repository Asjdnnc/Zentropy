# Phase 1: PQC Chat

Welcome to **Phase 1: PQC Chat**. This phase of the project focuses on integrating Post-Quantum Cryptography (PQC) into secure, cross-platform messaging clients. Our goal during this phase was to engineer custom solutions that ensure end-to-end quantum-resistant encryption for chat applications.

## Folder Structure

- `Quantum_Algos_MAC/`: Contains the PQC Chat client implementation specifically tailored for macOS.
- `Quantum_Algos_Windows/`: Contains the PQC Chat client and backend server logic built for Windows.
